# yarn1-listdup

List duplicate dependencies in yarn.lock file v1.

## Installation

Install globally with npm:

```bash
npm install -g yarn1-listdup
```

You can also execute it with `npx`:

```bash
npx yarn1-listdup
```

## Usage

Then you can run the command in the root of your Yarn 1 project:

```bash
yarn1-listdup
```
